Motivação A principal vantagem de usar o padrão MVC (acrônimo para Model View Controller) é a separação das responsabilidades que você tem entre as camadas. Cada camada possui sua responsabilidade referente a uma parte do trabalho da sua aplicação. Outra vantagem é a facilidade nas manutenções posteriores da sua aplicação, pois em aplicações onde apenas um arquivo “faz tudo” a manutenção se torna uma tarefa complicada e cara. Requisitos PHP 7.2 (e o drive pgsql) PostgreSQL Criando o Banco de dados Para que nossa aplicação funcione corretamente, devemos criar nosso banco de dados.
